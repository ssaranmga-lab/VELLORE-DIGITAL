How to Run This Website

This is a static HTML/CSS/JavaScript website that doesn't require a server to run. Here are three simple methods to view the website:

Method 1: Double-Click on index.html

  1. Open File Explorer
  2. Navigate to D:\MY PROJECT 2\MARKETING
  3. Double-click on index.html
  4. The website will open in your default web browser
Open index.html

Method 2: Drag and Drop into Browser

  1. Open File Explorer
  2. Navigate to D:\MY PROJECT 2\MARKETING
  3. Open your web browser (Chrome, Firefox, Edge, etc.)
  4. Drag the index.html file and drop it into the browser window

Method 3: Use File Protocol URL

  1. Open your web browser
  2. In the address bar, type: file:///D:/MY%20PROJECT%202/MARKETING/index.html
  3. Press Enter

Note: The net::ERR_CONNECTION_REFUSED error occurs when trying to access http://localhost:8080 because there's no server running on that port. The methods above don't require a server and will work correctly.
